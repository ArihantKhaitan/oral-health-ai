# Model Files

Place your trained model files here:

1. `oral_disease_model.h5` - The trained TensorFlow/Keras model
2. `class_names.json` - Already created with class labels

## How to get the model:

1. Go to your Kaggle notebook
2. Download from the Output section
3. Place the `.h5` file in this folder

## Model Details:
- Architecture: EfficientNetB0
- Input Size: 224x224x3
- Output: 8 classes (softmax)
- Size: ~50 MB
